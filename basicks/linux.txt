cd test - переходим в папку test (если это папка она существует)
cd ..  - переход в родительскую папку 
cd - переход в домашнюю папку 
cd ~   -переход в домашнюю папку 
mkdir - создание новой папки
mkdir test/new_dir - создание новой папки new dir уже в существующей папке тест 
touch test.txt - создание файл test.txt 
rm test.txt - удаление файла
rm -rf test - удаление папку со всем ее содержимым
ls -a  - просмотр всех файлов даже скрытых 
--help и выйдет руководство по исп по команде
